services

services library

Globally-accessible application services, such as cache management, user authentication, input validation, etc.

All of the service class instances are implemented as subclasses of the GsaService object.

Classes

GsaService
This abstract class defines a globally-accessible service with various Flutter APIs such as currency conversion, caching, logging, user authentication, and internationalization.
GsaServiceAppTrackingTransparency
The AppTrackingTransparency framework presents an app-tracking authorization request to the user and provides the tracking authorization status.
GsaServiceAuth
GsaServiceBookmarks
GsaServiceCache
Cache data manager, implemented with the shared_preferences package.
GsaServiceConsent
Class responsible for handling any user privacy-related consent requests and statuses.
GsaServiceLocation
Linter services with custom code inspection configuration.
GsaServiceNavigator
Designed for application navigation services implementation and handling.
GsaServiceTracking
User activity tracking service.
GsaServiceUrlLauncher
Service responsible for opening external apps via specified addresses.

Extensions

GsaServiceCacheEntryExt
Extension methods for the cached data values, including methods for retrieving and storing of data to the device memory.

Enums

GsaServiceCacheEntry
Identifiers (keys) for cached values.
